🎯 使用说明

在 Phaser.js 中使用：

// 加载精灵表
this.load.spritesheet('cards', 'assets/cards.png', {
    frameWidth: 70,
    frameHeight: 95
});

// 加载卡背
this.load.image('card_back', 'assets/card_back.png');

// 使用特定卡牌（例如：第一张卡牌）
this.add.sprite(x, y, 'cards', 0);

卡牌索引对照：

  • 帧 0-51：第一副牌
  • 帧 52-103：第二副牌
  • 帧 104-105：小王×2
  • 帧 106-107：大王×2
